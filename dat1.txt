HOW TO UPLOAD ONLY THE 'whatsapp' FOLDER TO GITHUB
===================================================

Follow these steps in your terminal to upload just the whatsapp folder to your new repository (https://github.com/dezignmvs/festnotify.git).

STEP 1: Open your terminal and navigate to the whatsapp folder inside your project
cd e:\zuh.ail__\2026\codes\Festival\fest_managment-main\whatsapp

STEP 2: Initialize a new Git repository (this creates a separate Git repo just for this folder)
git init

STEP 3: Add all files in the whatsapp folder to staging
git add .

STEP 4: Commit your changes
git commit -m "Initial commit for WhatsApp bot"

STEP 5: Add your GitHub repository as the remote origin
git remote add origin https://github.com/dezignmvs/festnotify.git

(Note: If you get an error saying "remote origin already exists", run this command instead to update it: 
git remote set-url origin https://github.com/dezignmvs/festnotify.git)

STEP 6: Rename the default branch to 'main' (this is the standard for GitHub)
git branch -M main

STEP 7: Push the code to GitHub!
git push -u origin main

---
Troubleshooting:
- If you get an authentication error when pushing, GitHub might prompt you to log in via your browser.
- If it says "Updates were rejected because the remote contains work that you do not have", it means your GitHub repo already has files (like a README). You can force push by adding -f to the last command: `git push -u origin main -f` (Warning: this overwrites whatever is currently on the GitHub repo).
